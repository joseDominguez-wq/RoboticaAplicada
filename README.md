# Prácticas con diferentes microcontroladores

Este repositorio contiene una serie de prácticas, enfocadas en el uso de **PWM (Pulse Width Modulation)** para el control de un LED, un motor DC y servomotores de **180° y 360°**.

 Material utilizado
- Arduino (UNO/MEGA)
- ESP32
- Raspberrypi 3
- LED
- Resistencia (220Ω – 330Ω)
- Motor DC
- Servomotor 180°
- Servomotor 360°
- Protoboard
- Cables jumper
- Fuente de alimentación 

---

## Descripción de las prácticas

### Control de LED con PWM
Se utiliza una salida PWM del Arduino para variar la intensidad luminosa del LED, simulando un efecto de encendido gradual.

###  Motor DC con PWM
El motor DC es controlado mediante una señal PWM para regular su velocidad de giro y su dirección de giro.

###  Servomotor de 180°
El servomotor se posiciona en ángulos específicos entre 0° y 180°.

###  Servomotor de 360°
El servomotor gira de manera continua; el PWM controla la velocidad y el sentido del giro.

---

## Cómo ejecutar los códigos
1. Abrir el **Arduino IDE**
2. Conectar la placa del microcontrolador a la computadora
3. Abrir el archivo correspondiente
4. Seleccionar la placa y el puerto correctos
5. Cargar el programa

---

## Notas
- Algunos componentes pueden requerir **alimentación externa**.
- Se debe revisar correctamente las conexiones antes de energizar el circuito.
- Los códigos son con fines **educativos y de práctica**.
